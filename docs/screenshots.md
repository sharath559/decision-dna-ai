# DecisionDNA AI — Screenshots Guide

## How to Capture Screenshots

After running `streamlit run app.py`, navigate to each tab and take screenshots:

### 1. Dashboard
- URL: `http://localhost:8501` (Dashboard tab)
- Shows: Total cases, drift count, risk count, mutation scores bar chart, summary table

### 2. Case Explorer
- Click the "Case Explorer" tab
- Expand each case to show description, decisions, and risk level

### 3. Decision DNA View
- Select a case in sidebar → click "Decision DNA" tab
- Shows side-by-side old vs new genome comparison

### 4. Mutation Analysis
- Click "Mutation Analysis" tab
- Shows: Mutation score, confidence, primary mutation, root cause, recommendation, radar chart

### 5. Timeline View
- Click "Timeline" tab
- Shows: Temporal event timeline with color-coded categories

### 6. Evidence & Missing Documents
- Click "Evidence" tab
- Shows: Before/after evidence comparison, missing documents highlighted

### 7. Security Review
- Click "Security" tab
- Test 1: Default scan result (should show PASSED)
- Test 2: Enter malicious text and click Scan (should show BLOCKED with findings)

### 8. Audit Report
- Click "Audit Report" tab
- Shows: Report ID, executive summary, lineage, root cause, impact, originality metadata

## Recommended Screenshot Resolution

- Browser width: 1280px or wider
- Use full-screen mode for cleaner captures
- Dark mode or light mode both work

## For Video

Record the full browser window while navigating through all eight tabs.
Use the sidebar case selector to switch between the three demo cases.
End with the security scanner demo.
